# offtrailrun.com

I don’t expect many people to read this, but if you’re here — hi 👋

This is the code for [offtrailrun.com](https://offtrailrun.com), a small website to introduce runners in Eindhoven to orienteering.

The idea is simple:

- Get a map.
- Find the controls.
- Choose your own route.

Most people in the Netherlands do not really know what orienteering is. Runners understand trail running, but “orienteering” can sound more technical than it is.

So this site makes the first step easier: start with an off-trail run in a park, then try a real orienteering training in the forest.

## About the site

The site has two parts:

- `/` — the main landing page for people who may want to join a run
- `/start` — a beginner guide that explains the basics: map symbols, compass use, contour lines, and route choice

It also supports English and Dutch:

- `/`
- `/nl/`
- `/start`
- `/nl/start/`

## Tech

Built with:

- Astro
- Tailwind CSS
- Netlify
- EN/NL translation files
